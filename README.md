# KampüsBul - Kayıp Eşya Bulma Uygulaması

Kampüs içinde kayıp olan eşyaların fotoğraflarının çekilerek açıklama ve konum bilgileri ile birlikte uygulamaya eklenmesiyle kayıp eşyaların sahibine ulaştırılmasını hedefleyen bir mobil uygulama.

## 📱 Özellikler
- Google/Firebase Auth ile Güvenli Giriş
- Fotoğraflı İlan Verme
- Gerçek Zamanlı Sohbet
- Kategori ve Tarih Bazlı Filtreleme
- Bildirim Merkezi
- Profil ve Şifre Yönetimi
